🎄 Christmas Scripts

Beautiful, lightweight JavaScript animations to bring holiday cheer to your website

Transform your website into a winter wonderland with these delightful Christmas-themed JavaScript animations! Each script is designed to be lightweight, easy to integrate, and customizable with URL parameters. Simply include the script tag and watch the magic happen! ✨

❄️

Snowflakes

Create beautiful falling snowflakes across your entire page with automatic theme detection.

📋 Parameters:

startDate When to start showing snowflakes (MM-DD format) Default: "12-01"
endDate When to stop showing snowflakes (MM-DD format) Default: "01-15"
theme Color scheme: "light", "dark", or null for auto-detection Default: auto-detect
<script src="christmas-snowflakes.js?startDate=11-01&endDate=01-10&theme=dark"></script>
🎀

Garlands

Decorate your page with festive garlands that gently sway and breathe.

📋 Parameters:

startDate When to start showing garlands (MM-DD format) Default: "12-01"
endDate When to stop showing garlands (MM-DD format) Default: "01-15"
position Where to place garlands: "top", "bottom", or "both" Default: "both"
<script src="christmas-garlands.js?startDate=11-01&endDate=01-10&position=both"></script>
🎅

Santa Claus

Watch Santa fly across your screen in his sleigh, spreading holiday joy!

📋 Parameters:

startDate When to start Santa's flights (MM-DD format) Default: "12-01"
endDate When to stop Santa's flights (MM-DD format) Default: "01-15"
onlyHorizontalMovements Set to "true" or "1" for horizontal-only flight path Default: false (diagonal movement enabled)
<script src="christmas-santa-claus.js?startDate=11-01&endDate=01-10&onlyHorizontalMovements=false"></script>
💡

Christmas Lights

Add colorful twinkling lights around the borders of your webpage.

📋 Parameters:

startDate When to start showing lights (MM-DD format) Default: "12-01"
endDate When to stop showing lights (MM-DD format) Default: "01-15"
theme Color scheme: "light", "dark", or null for auto-detection Default: auto-detect
positions Comma-separated positions: "top", "right", "bottom", "left" Default: "top,right,bottom,left"
<script src="christmas-lights.js?startDate=11-01&endDate=01-10&positions=top,bottom&theme=dark"></script>

🎁 Made with love and holiday spirit • Customize dates and themes via URL parameters • Enjoy the festive season! 🎄
